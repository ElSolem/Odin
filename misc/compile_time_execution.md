## Symbols
- `!` = Negate Signal/Address
- `@` = Context Signal/Address
- `#` = Type Address/Signal
- `$` = String Address/Signal
- `%` = Return Address/Signal
- `^` = Memory Reference/Address  
- `&` = Memory Reference/Signal  
- `?` = Value Address/Reference
